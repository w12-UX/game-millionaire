# 一掷千金 Deal or No Deal

复刻美国经典综艺《Deal or No Deal》核心玩法的轻量化前端小游戏。

- **在线体验**: https://w12-ux.github.io/game-millionaire/
- **技术栈**: 纯前端 HTML + CSS + JavaScript（零后端）
- **部署**: GitHub Pages

---

## 项目结构

```
game-millionaire/
├── index.html              # 主入口
├── assets/
│   ├── css/
```

### 银行家报价算法

1. **轮次系数**: 前期 10-30% / 中期 40-60% / 后期 70-90%
2. **大额淘汰修正**: 每开出 ≥$100k，系数下调 12%
3. **专属箱兜底**: 大额抬 10% / 小额压 20%
4. **微幅浮动**: ±5%

## 如何继续开发

```bash
# 克隆项目
git clone https://github.com/w12-UX/game-millionaire.git
cd game-millionaire

# 本地测试（用任意 HTTP 服务器）
python -m http.server 3000
# 浏览器打开 http://localhost:3000

# 修改后推送
git add .
git commit -m "feat: 你的改动"
git push origin main
# GitHub Pages 自动重新部署
```

> **版本标记**: `第一版` 标签记录了当前稳定版本。
> `git checkout 第一版` 可回到此版本。
